# Fallout-2d20-windows
Application Fallout 2d20 character pour windows sans installation 
📥 INSTALLATION

1️⃣ Téléchargez le fichier .zip
2️⃣ Faites clic droit → “Extraire tout…”
3️⃣ Ouvrez le dossier extrait
4️⃣ Double-cliquez sur le fichier .exe pour lancer l’application
